# Stopwatch and timer

## An always on screen timer and stopwatch with millisecond precision for MacOS built using Swift and UIKit.

### Set your preferences by clicking the ⚙️ button:
<p align="center">
<img src="https://i.imgur.com/xm7egH8.png" alt="Stopwatch/timer user interface" height="85.93%" width="50.61%" hspace="30"/>
</p>

### When always on top and transparency are enabled only the clock will appear on screen
<p align="center">
<img src="https://i.imgur.com/4J7wolv.png" alt="Stopwatch/timer user interface" height="85.93%" width="50.61%" hspace="30"/>
</p>

1. Run and pause the <b>stopwatch</b> by clicking the ⏯ button.<br>
2. Insert a time and run and pause the countdown <b>timer</b> by clicking the ⏯ button.
3. Reset both the stopwatch or timer by clicking the ◼ button.
